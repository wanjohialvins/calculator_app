# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

- Added keyboard support for numbers and operations
- Improved calculator layout for better usability
- Enhanced display of calculation history
- Added CSV export for calculation history
- Updated CSS to align 0, ., =, + on the same row
- Improved responsive design for smaller screens

## [1.0.0] - 2025-11-27

### Added
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Calculator display panel
- History panel with calculation logging
- Buttons for numbers 0–9, decimal point, operations (+, −, ×, ÷)
- Clear (C), Toggle Sign (±), Percent (%), Equals (=)
- Responsive design for desktop and mobile
- Export history to CSV functionality

### Changed
- UI styling and colors
- Grid layout for buttons

### Fixed
- Display issues with multi-digit input
- Division by zero handling

## [0.9.0] - 2025-11-26

### Added
- Initial working calculator with button clicks
- Basic JavaScript logic for computations
- History panel with “No calculations yet” message
