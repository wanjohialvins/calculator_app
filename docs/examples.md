# Calculator Examples

This document demonstrates how to use the Eragon Calc application with examples of basic operations, history usage, and keyboard shortcuts.


## 1. Performing Basic Operations

### Addition
Click buttons or use the keyboard to add numbers.

**Example:**

Input: 7 + 3 =
Display: 10

### Subtraction
Input: 9 - 4 =
Display: 5

### Multiplication
Input: 6 × 5 =
Display: 30

### Division
Input: 12 ÷ 4 =
Display: 3


**Note:** Division by zero will show `Error`.

---

## 2. Using Additional Buttons

- **C (Clear):** Resets the current input.
- **± (Toggle Sign):** Changes the sign of the current number.
- **% (Percent):** Converts the current number to a percentage.
- **= (Equals):** Computes the result of the current operation.

---

## 3. Keyboard Shortcuts

| Key          | Action                  |
|--------------|------------------------|
| 0–9          | Enter number           |
| `.`          | Decimal point          |
| `+`          | Addition               |
| `-`          | Subtraction            |
| `*` or `x`   | Multiplication         |
| `/`          | Division               |
| `Enter` or `=` | Compute result        |
| `Escape`     | Clear                  |

---

## 4. History Usage

The calculator stores all completed operations in a **history panel**.

**Example:**

1. Perform `5 + 3 =` → Result `8`  
2. Perform `10 ÷ 2 =` → Result `5`  

The history panel will show:

5 + 3 → 8
10 ÷ 2 → 5


### Export History
Click the **Export** button to download the history as a `.csv` file:

5,+,3,8
10,/,2,5


### Clear History
Click the **Clear History** button to remove all stored calculations.
