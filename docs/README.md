 # Eragon Calc Project

A fully functional calculator application built in JavaScript for a hypothetical operating system. This project simulates a standard calculator capable of performing basic operations and keeping track of all calculations made, with history logging and export functionality.

---

## Features

- Addition, subtraction, multiplication, and division.
- Calculation history log.
- Export history to `.csv`.
- Keyboard support for numbers and operations.
- Clear, toggle sign, and percentage functionality.
- Professional and responsive UI design.

## Usage

1. Open `index.html` in your preferred web browser.
2. Use the on-screen buttons or keyboard to perform calculations.
3. Check the calculation history on the right panel.
4. Export the history to CSV if needed.
5. Clear the display or history as needed using the buttons.

---

## Keyboard Support

- Numbers: `0–9`
- Decimal point: `.`
- Operations: `+`, `-`, `*` or `x`, `/`
- Equals: `Enter` or `=`
- Clear: `Escape`
- Toggle sign: `±`
- Percent: `%`

---

## History Panel

- Displays all past calculations.
- Shows expression and result.
- Clear or export history as required.
