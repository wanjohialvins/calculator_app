# Eragon Calc — Installation Guide

This document explains how to set up and run the Eragon Calc application on your machine.

---

## 1. Prerequisites

Before installing the calculator, ensure you have the following:

- A modern web browser (Chrome, Firefox, Edge, or Safari)
- A text editor or IDE (optional, e.g., VS Code) if you want to modify the code
- Git (optional, for cloning the repository)

---

## 2. Download the Project

You can get the project files in two ways:

### Option 1: Clone the Repository

```bash
git clone https://github.com/your-username/calculator_app.git
cd calculator_app
Option 2: Download ZIP
Go to the repository page.

Click Code → Download ZIP.

Extract the ZIP file to your preferred folder.

3. Running the Calculator
Option 1: Open in Browser
Navigate to the project folder.

Open index.html in your web browser.

The calculator should appear and be fully functional.

Option 2: Serve via Local Web Server (Optional)
If you have VS Code:

Install the Live Server extension.

Right-click index.html and select Open with Live Server.

Your calculator will run in the browser with live reloading.

4. Using the Calculator
Click buttons to enter numbers and operators.

Keyboard input is supported:

Numbers: 0–9

Decimal: .

Addition: +

Subtraction: -

Multiplication: * or x

Division: /

Equals: Enter or =

Clear: Escape

View your calculation history in the side panel.

Export history to a .csv file if needed.

5. Optional Steps
Customize the CSS in style.css for different themes.

Modify script.js to add advanced operations.

Keep the project under version control using Git.

6. Troubleshooting
Calculator not showing: Ensure index.html is opened in a browser.

Buttons unresponsive: Make sure script.js is in the same folder and properly linked.

History export fails: Ensure you allow downloads in your browser.

7. Support
For questions or issues, contact the repository owner or open an issue on GitHub.
